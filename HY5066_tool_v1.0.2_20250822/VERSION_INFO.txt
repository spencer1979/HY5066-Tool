HY5066 Bluetooth Control Tool - 版本資訊

功能特色 / Features:
- HY5066 藍牙模組控制 / HY5066 Bluetooth Module Control

- AT 指令發送與接收 / AT Command Send & Receive

- 序列埠自動偵測 / Auto Serial Port Detection

- Windows 7 或更新版本 / Windows 7 or later
- 支援藍牙的 HY5066 模組 / HY5066 Module with Bluetooth support
- USB 轉序列埠連接線 / USB to Serial Connection Cable

2. 雙擊「啟動 HY5066 控制工具.bat」
3. 或直接執行 HY5066_tool.exe
